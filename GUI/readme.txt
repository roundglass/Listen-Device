Steps to convert .py to .exe

1. Open cmd window and type 'pip install pyinstaller'
2. Open where python File is located in our case go inside Source folder
3. Press Shift and Right click at the Same location.
4. Click on ‘Open PowerShell window here’. 
5. Now type pyinstaller --listenConfig -w 'ListenConfig.py'
